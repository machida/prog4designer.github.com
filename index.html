P4D とは？

デザイナーとプログラマーで集まって Web や アプリをつくったりデザインやプログラミングについて相談したりする会です。

現在は毎月 2 回ずつ都内で meetup を開いています。

興味のある方はぜひお気軽に Facebook グループ へご加入ください。

対象

  • プログラム (Ruby, Rails, JavaScript など) や Git もできるようになりたいデザイナ
  • デザインも出来るようになりたいプログラマ
